### Tools

- Vue3 / NuxtJS
- Nuxt API Routes
- TailwindCSS / shadcn-Vue
- AuthJS / Auth0: Authentication integration with Google and Github Account
- Pinia: State Management
- MongoDB: Main Database
- Firebase: Image Storage
- Zod: Form Validation
